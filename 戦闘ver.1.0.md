int STR
int VIT

int HP
int LV

データベースから受け取った自分のパートナーのexp情報（個人）と
各々の設定したキャラクターのbaseパラメータ値とperLv値を取得しexpを用いてパラメータを計算する

例　STR = baseSTR + Lv*perLvSTR
    Lv = (SQRT(exp)+4)/7

#対戦相手選択画面

選択画面に映った後対戦相手の情報を取得パートナーの名前と相手のプレイヤーの名前,レベルを表示する（この際ほかのパラメータの値を読み込むか、対戦画面に映る前に読み込むかは未定）

この状態で対戦相手をタップすると次の画面へ遷移する


#対戦画面

この画面に遷移したときに戦闘の計算を終わらせておく

画面上部に自分と相手の使用パートナーの画像と名前とレベルを表示する

□
  vs
　　□
配置は配置してから見る必要がある

画面の下半分（３割ほどかも）はメッセージダイアログ画面となる
そのメッセージダイアログにターン制の行動を記載し、タップするたびにログが送られる
勝利または敗北すると結果が全画面に表示されて戦闘終了となる

以下計算式
攻撃側の能力値を1、防御側の能力値を2とする

double ad = str1*crt*rand*100*avoid/(vit2+100);
double avoid = agi2 + luk2/2 - luk1*1.2
double rand = 0.8,0.9,1.0,1.1,1.2の乱数
ダメージの表示は整数で行う

行動順はAGIの高い方を先行として交互に行う
攻撃演出などは余裕があったら
例

ジバニャン　vs  デデンネ

1ターン目

ジバニャンの攻撃
	デデンネは10ダメージを喰らった
					▼
2ターン目

デデンネの攻撃
	ジバニャンは999999ダメージを喰らった
					▼
ジバニャンは倒れた
					▼
